# 049. 三色标记和写屏障

## 问题

三色标记、写屏障和 STW 分别是什么？

## 核心答案

三色标记把对象分为白、灰、黑三类。白色表示尚未标记，灰色表示已发现但子对象未扫描完，黑色表示自身和子对象都已扫描。

并发 GC 期间程序仍在修改指针关系，写屏障用于维护标记正确性，避免活对象被错误回收。

STW 是 stop-the-world，表示短暂停止用户 goroutine。Go GC 尽量把大部分工作并发化，但仍有少量阶段需要 STW。

## 代码示例

```bash
GODEBUG=gctrace=1 go test ./...
```
