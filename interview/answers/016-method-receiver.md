# 016. 值接收者和指针接收者

## 问题

值接收者和指针接收者应该怎么选？

## 核心答案

值接收者会复制接收者，适合小对象、不可变语义或不需要修改状态的场景。

指针接收者可以修改原对象，避免复制大对象，也能保证方法集合一致。只要某些方法需要指针接收者，通常建议这个类型的方法都使用指针接收者，避免接口实现上的混乱。

## 代码示例

```go
type Counter struct {
	n int
}

func (c *Counter) Inc() {
	c.n++
}
```
