# 001. 数组和切片

## 问题

Go 中数组和切片有什么区别？切片扩容时需要注意什么？

## 答案

数组是定长值类型，长度属于类型的一部分，例如 `[3]int` 和 `[4]int` 是不同类型。数组赋值或传参会复制整个数组。

切片是对底层数组的引用视图，包含指针、长度和容量。切片赋值或传参只复制切片头，不复制底层数组。

切片扩容时，`append` 可能复用原底层数组，也可能分配新数组。如果多个切片共享同一个底层数组，修改其中一个切片可能影响另一个切片。

## 代码示例

```go
package main

import "fmt"

func main() {
	a := []int{1, 2, 3}
	b := a[:2]

	b = append(b, 100)

	fmt.Println(a) // [1 2 100]
	fmt.Println(b) // [1 2 100]
}
```
