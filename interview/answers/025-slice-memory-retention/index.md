# 025. 切片内存滞留

## 问题

大切片截取小切片为什么可能导致内存不释放？

## 核心答案

切片只是底层数组的视图。小切片如果仍然引用大数组的一部分，整个底层数组都可能无法被 GC 回收。

如果只需要保留少量数据，应该复制出新的切片，让旧的大数组失去引用。

## 代码示例

```go
small := append([]byte(nil), big[:10]...)
big = nil
```
