# 013. 函数、方法、闭包和可变参数

## 问题

Go 中函数、方法、闭包和可变参数分别有什么特点？

## 核心答案

函数是一等值，可以赋值给变量、作为参数传递或作为返回值。

方法是带接收者的函数，接收者可以是值或指针。

闭包会捕获外部变量，变量生命周期可能因此延长。

可变参数本质上在函数内部表现为切片。

## 代码示例

```go
func sum(nums ...int) int {
	total := 0
	for _, n := range nums {
		total += n
	}
	return total
}
```
