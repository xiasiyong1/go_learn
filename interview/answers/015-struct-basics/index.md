# 015. 结构体基础

## 问题

结构体值拷贝、匿名字段和 tag 分别怎么理解？

## 核心答案

结构体是值类型，赋值和传参默认会复制整个结构体。结构体里如果包含 slice、map、指针等引用型字段，复制的是字段值，底层数据仍可能共享。

匿名字段用于组合，Go 通过组合实现代码复用，而不是传统继承。

tag 是字段上的元信息，常用于 JSON、数据库 ORM 和校验框架。

## 代码示例

```go
type User struct {
	ID   int    `json:"id"`
	Name string `json:"name"`
}
```
