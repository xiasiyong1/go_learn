# 046. interface 底层实现

## 问题

interface 底层表示是什么？空接口和非空接口有什么区别？

## 核心答案

接口值通常可以理解为“类型信息 + 数据指针”。空接口不要求方法集，只需要记录动态类型和值。非空接口还需要记录和方法集相关的信息，用于动态派发。

接口为 nil 的条件是类型信息和值都为 nil。只要有动态类型，即使动态值是 nil，接口也不等于 nil。

## 代码示例

```go
var p *MyError = nil
var err error = p
fmt.Println(err == nil) // false
```
